## Redis

- Distributed in-memory data store(memcache와 유사)
- 자료구조들을 저장하기 좋음
- 데이터를 디스크로 저장가능
- 데이터 스토어로도 사용가능
- web apps에 대한 caching layer로 많이 사용됨

## Command

- ZRANK
    - `ZRANK key member` : score를 기준으로 오름차순으로 정렬했을때, key에 있는 member의 순위를 반환
