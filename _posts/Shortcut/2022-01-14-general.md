## Intellij

| 동작 | 커멘드 
| -- |
| reformat code | Option+Cmd+L
| finding file | Cmd+Shift+O
| searching everywhere | press Shift twice
| Replace text in the current file | Ctrl + R
| Run all tests | `Ctrl + Shift + R`
